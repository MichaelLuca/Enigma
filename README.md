## Lastenheft für das Parkhaus (Brainstorming)
Das System DigiPark
- Muss
	- die Anzahl der Autos und Stellplätze muss vom Schrankenwärter gekannt werden um den Zugang zum Parkhaus zu gewähren.
	- das Parkhaus muss die Startzeit des Parkens für einen Parkausweis erfassen
	- Der Kassierer muss die Parkgebür anhand der Startzeit des Parkausweises berechnen, es sei denn es liegt ein Ausgewiesener Notfall vor (Feuer, Hausalarm).
	- Der Kassierer muss den Parkausweis nach Bezahlung entwerten.
	- Der Schrankenwärter muss den Parkenden bis zu 15 Minuten nach Entwärtung den Parkenden aus dem Parkhaus fahren lassen.
	- Der Manager muss den Kassenstand pro Tag, pro Woche, pro Monat und pro Jahr kennen und in einer Kurve eintragen.
	
- Sollte
	- der Einweiser sollte den nächsten freien Platz im Parkhaus nehmen.
	- Der Manager sollte wissen, wie viele Stellplätze gerade frei sind und kennt den Kassenstand.
	- Das Parkhaus sollte nicht überfüllt werden können.
	- Im Notfall sollten alle Schranken geöffnet werden können.
	
### Funktional (Was das Produkt können soll)
- Das System DigiPark muss Autos das Parken und Herausfahren mit Kontrollen gewähren
- Das System DigiPark muss die Fülle des Parkhauses verwalten
- Das System DigiPark muss im Notfall alle Schranken sofort öffnen
- Das System DigiPark sollte die Parkgebühr anhand der Anfangsparkzeit errechnen
- Das System DigiPark sollte den Herausfahrenden nur nach Bezahlung des Parkausweises passieren lassen
